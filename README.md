## 프로젝트 소개

본 리포지터리는 경영경제 소프트웨어 프로그래밍 기말대체과제를 위해 제작한 HTML 기반 웹앱을 포함하고 있습니다.

서버를 사용하지 않고 브라우저에서 실행되는 단일 HTML 파일 형태로 제작하였으며, GitHub Pages를 통해 웹에서 직접 실행할 수 있도록 게시하였습니다.

## 포함된 앱

### 1. PDF 편집 앱

PDF 파일을 업로드한 뒤 필요한 페이지만 남기고, 페이지 순서를 변경하여 새로운 PDF 파일로 다운로드할 수 있는 앱입니다.

주요 기능은 다음과 같습니다.

* 하나 또는 여러 개의 PDF 파일 업로드
* PDF 페이지 목록 및 미리보기 표시
* 불필요한 페이지 삭제
* 페이지 순서 변경
* 편집된 PDF 파일 다운로드

실행 링크:

[PDF 편집 앱 실행하기](https://primaveraforgood-prog.github.io/final-2026/pdf-editor.html)

파일명:

```text
pdf-editor.html
```

### 2. 이미지 자르기 앱

JPG, PNG 등 이미지 파일을 업로드한 뒤 원하는 가로세로 비율로 이미지를 자르고, 새로운 이미지 파일로 다운로드할 수 있는 앱입니다.

주요 기능은 다음과 같습니다.

* JPG, PNG, WEBP 이미지 파일 업로드
* 업로드한 이미지 화면 표시
* 다양한 비율로 이미지 자르기

  * 1:1
  * 4:3
  * 16:9
  * 3:4
  * 9:16
  * 자유 비율
* 자른 이미지 미리보기
* 자른 이미지 파일 다운로드

실행 링크:

[이미지 자르기 앱 실행하기](https://primaveraforgood-prog.github.io/final-2026/image-cropper.html)

파일명:

```text
image-cropper.html
```

## 실행 방법

1. GitHub Pages 홈페이지에 접속합니다.
2. PDF 편집 앱 또는 이미지 자르기 앱 링크를 선택합니다.
3. 브라우저에서 파일을 업로드합니다.
4. 필요한 편집 작업을 진행합니다.
5. 결과 파일을 다운로드합니다.
